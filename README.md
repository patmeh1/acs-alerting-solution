# AF Group Critical Alerting Solution

Single-page GitHub Pages site describing a Microsoft-native replacement architecture for VTEXT-based production support alerts.

The site recommends Azure Monitor Action Groups for SMS and voice notification, with Logic Apps or Azure Functions for event ingestion and routing. It includes an updated use-case analysis, architecture diagram, cost model, implementation details, short-term and long-term plans, open questions, alternative options, and a detailed step-by-step build guide with sample scripts.
